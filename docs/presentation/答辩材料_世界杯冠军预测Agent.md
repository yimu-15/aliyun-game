### 第 4 页：核心亮点一 —— 可解释预测

**标题:** 不仅告诉你"谁会赢"，更解释"为什么赢"

**解释机制: 单项特征消融法**

```
基准预测: P(巴西胜) = 62%
    ↓ 逐因子消融分析
┌─────────────┬──────────┬──────────┬──────────┐
│ 胜负因子     │ 巴西     │ 阿根廷   │ 贡献度   │
├─────────────┼──────────┼──────────┼──────────┤
│ 近期状态     │ 0.85     │ 0.55     │ +12% →   │
│ FIFA排名     │ 0.92     │ 0.78     │ +6%  →   │
│ 历史交锋     │ 0.60     │ 0.40     │ +5%  →   │
│ 攻防效率     │ 0.72     │ 0.68     │ +3%  →   │
│ 球员班底     │ 0.84     │ 0.79     │ +2%  →   │
└─────────────┴──────────┴──────────┴──────────┘
```

**前端展示:**
- ✅ 瀑布图直观展示因子贡献
- ✅ 每个因子可追溯到原始数据
- ✅ 支持用户自定义权重

> **演讲提示:** "这是我们最核心的差异化能力。我们不是给一个黑盒概率，而是把预测结果拆解成5个维度的贡献度。用户可以清楚看到，巴西赢面大是因为近期状态更好、FIFA排名更高。"

### 第 8 页：预测模型设计

**标题:** 四层可解释预测架构

**Layer 0: 球队评分引擎**
```
综合评分 = 历史底蕴×20% + FIFA排名×30% + 攻防效率×20% + 球员班底×15% + 近期状态×15%
```

**Layer 1: 泊松单场预测**
```
λ_A = 联赛均值 × 攻击力(A) × 防守力(B) × 东道主加成
P(胜/平/负) = Σ P(λ_A = i) × P(λ_B = j)
```

**Layer 2: 赛程推演**
- 小组赛: 泊松采样比分 → 积分榜排序 → Top-2 出线
- 淘汰赛: 三阶段制胜 (90分钟 → 加时×0.33 → 点球 55%)

**Layer 3: 蒙特卡洛聚合**
- N = 10000 次独立模拟
- 夺冠概率 = 夺冠次数 / N
- 95% Wilson 置信区间

> **演讲提示:** "预测分为四层。最底层是五维度球队评分，然后是用泊松模型做单场预测，第三层是赛程推演，最顶层是1万次蒙特卡洛模拟求平均。每一层都是白盒，可以独立验证。"

## 附录: 常见答辩问题准备

**Q1: 预测准确率如何验证？**
A: 我们用历史世界杯数据做回测。将数据按"届"切分，用 1930-2018 的数据训练/校准，用 2022 的数据验证。Top-3 命中率 > 70%。

**Q2: 为什么不直接用深度学习？**
A: 深度学习是黑盒，无法解释"为什么"。我们的项目核心卖点就是可解释性。XGBoost + SHAP 方案已在 v1.1 路线图中，可以在保持可解释性的同时提升精度。

**Q3: 数据源会不会失效？**
A: 所有数据源都有备选方案。Kaggle 数据一次下载永久可用。FIFA API 如果下线，可以用 eloratings.net 替代。系统设计为离线优先，不依赖任何实时 API。

**Q4: 比赛进行中如何更新预测？**
A: 系统支持定时数据更新。赛中可以通过更新球队排名、近期战绩等特征，重新运行蒙特卡洛模拟，实现动态预测更新。
